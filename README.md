# A-B-Testing---Marketing-Data-Anallysis

This project performs A/B testing to compare the performance of Control and Test marketing campaigns.

**Goal**:
Identify which campaign performs better in terms of user engagement and conversions

**Approach**:

1. Cleaned and validated data (missing values, duplicates)
2. Created key metrics like CTR, CPC, CAC, Conversion Rate
3. Analyzed user funnel (Impressions → Clicks → Purchase)
4. Compared both campaigns using visualizations
